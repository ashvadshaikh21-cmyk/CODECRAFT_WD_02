# CODECRAFT\_WD\_02

Advanced Stopwatch Web Application



Overview

An interactive stopwatch web application built using HTML, CSS, and JavaScript. It allows users to measure time accurately and record lap intervals.



Features

\- Start, Pause, Reset functionality

\- Lap time tracking

\- Smooth animated buttons

\- Gradient background

\- Dark/Light mode toggle

\- Clean UI with glassmorphism effect



Technologies Used

\- HTML5

\- CSS3 (Flexbox, Variables, Media Queries)

\- JavaScript (DOM Manipulation, setInterval)
